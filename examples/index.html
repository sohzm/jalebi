Jalebi is just a collection of web components that I use in my projects. For more information, visit github.com/sohzm/jalebi.




Marquee

Hello, world! This is an infinite marquee. Enjoy scrolling text! 
<jalebi-marquee direction="to-left" duration="10s">
    <span style="padding-right: 2rem">Hello, world!</span>
    <span style="padding-right: 2rem">This is an infinite marquee.</span>
    <span style="padding-right: 2rem">Enjoy scrolling text!</span>
</jalebi-marquee>

Tooltip
