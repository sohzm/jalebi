Jalebi is just a collection of web components that I use in my projects. For more information, visit github.com/sohzm/jalebi.

Jalebi isn't a framework or a library. It's just a bunch of web components that you can use in your projects. The components are built with vanilla JavaScript and have no external dependencies. This is a wip demo page for the components.

Include the component with: <script src="https://jalebi.soham.sh/src/jalebi-<component name>.js"></script>

Key Features

  • Lightweight web components
  • No dependencies
  • Customizable with CSS variables
  • Accessible and responsive

How to use

Import the component scripts and start using them in your HTML:

<script src="/src/jalebi-tabs.js"></script>

That's it! Now you can use the jalebi-tabs component.

Accordion

Jalebi is a collection of lightweight, customizable web components that can be easily integrated into any web project. The components are built with vanilla JavaScript and have no external dependencies.

You can include Jalebi directly from a CDN or download the source files:

<script src="https://cdn.example.com/jalebi.min.js"></script>

Or use npm:

npm install jalebi-components

Yes! Jalebi uses CSS variables that can be easily overridden to match your project's design system. This makes it simple to customize colors, fonts, spacing, and more.

For example:

:root { --fg-accent: #ff5500; --bg-accent: #fff1e6; }
